# GRAFCET File

The `elevator.grafcet` file is a GRAFCET-Studio v2.0 project file (XML format).

To view it:
1. Download [GRAFCET-Studio](https://www.grafcet-studio.de/)
2. Open the `.grafcet` file directly in the application

The file contains the complete sequential function chart (SFC/GRAFCET) specification for the 4-floor elevator control logic, including all steps, transitions, and I/O symbol mappings.

> Note: Upload the `elevator.grafcet` file manually to this folder via GitHub or git — it is a binary/XML file too large to push via the API.
