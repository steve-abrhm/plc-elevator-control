# CODESYS Project Files

This folder contains the CODESYS project files for the elevator controller. Due to file size limitations, these need to be uploaded manually via git or the GitHub web interface.

## Files to upload

| File | Description |
|---|---|
| `lift_with_door.project` | CODESYS project — elevator with basic door logic |
| `lift_with_door_opena_and_close.project` | CODESYS project — final version with full open/close sequencing |
| `lift_with_door-AllUsers.opt` | Shared project options |

## How to open

1. Install [CODESYS](https://www.codesys.com/) v3.5 or later
2. Open the `.project` file directly from CODESYS IDE
3. Use the Device Simulator to run without physical hardware

## Note

Build artifacts (`*.bootinfo`, `*.compileinfo`, `*.precompilecache`) are excluded from this repo — they are auto-generated by CODESYS on compile.
